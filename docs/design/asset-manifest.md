# Isometric Asset Manifest

## 1. Environment Tiles
- **tile-floor-wood.png:** 64x32 rhombic, warm brown, pixel-art wood grain.
- **tile-floor-carpet.png:** 64x32 rhombic, dark teal, subtle fabric texture.
- **tile-floor-concrete.png:** 64x32 rhombic, light grey, industrial feel.
- **tile-void.png:** 64x32 rhombic, solid dark grey or black for edges.

## 2. Furniture (Isometric South-East)
- **furniture-desk-modern.png:** 64x64 sprite (approx. 2x1 grid size). Light wood top, grey metal legs.
- **furniture-desk-server.png:** 64x96 sprite. Tall server rack with blinking pixel lights.
- **furniture-chair-office.png:** 32x48 sprite. Blue padded chair.
- **furniture-vending-machine.png:** 48x80 sprite. Red, iconic retro design with pixelated "Drink" sign.
- **furniture-plant-potted.png:** 32x64 sprite. Simple green leafy plant in a terracotta pot.
- **furniture-computer-set.png:** 32x32 sprite. CRT-style monitor and keyboard for placement on desks.

## 3. Agent Sprites (4-Way Directional)
All agents require 4 directions (NE, SE, SW, NW).
- **agent-base-idle.png:** 24x32 sprite sheet. 2-frame animation.
- **agent-base-walk.png:** 24x32 sprite sheet. 4-frame animation.
- **agent-base-typing.png:** 24x32 sprite sheet. 2-frame loop of hands on keys.

## 4. Special Effects & Overlays
- **fx-fire-loop.png:** 32x48 sprite sheet. 6-frame stylized orange/yellow pixel flames.
- **ui-selection-ring.png:** 64x32 rhombic ring, yellow dashed line.
- **ui-bubble-speech.png:** Isometric-angled speech bubble with a pixelated tail.

## 5. UI Elements
- **ui-icon-agent-card.png:** 64x64 headshot placeholders for each agent role.
- **ui-button-pixel-red.png:** "Cancel/Stop" button in red pixel-art style.
- **ui-button-pixel-green.png:** "Start/Run" button in green pixel-art style.
